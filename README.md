🌙 Jornada de Luna
📖 Sobre o Projeto

Jornada de Luna é um jogo de plataforma 2D desenvolvido na Unity como projeto da disciplina de Game Development.

O jogo acompanha Luna em uma aventura por cenários mágicos, onde o jogador deve superar obstáculos, coletar cristais e avançar entre fases progressivamente mais desafiadoras.

🎮 Mecânicas Implementadas
Movimento lateral;
Pulo;
Sistema de coleta de cristais;
HUD com vidas e contador de cristais;
Transição entre fases;
Tela de vitória;
Animações do personagem;
Feedback visual na coleta dos itens.
🗺️ Fases

O jogo possui:

Level 1
Level 2
Level 3
Tela de Vitória

Cada fase apresenta novos obstáculos e exige maior domínio dos movimentos do personagem.

🛠️ Tecnologias Utilizadas
Unity 6
C#
Tilemap 2D
TextMeshPro
Animator
Rigidbody2D e Collider2D
🎨 Assets Utilizados

Os sprites e elementos visuais foram obtidos a partir de assets gratuitos, utilizados exclusivamente para fins educacionais.

⭐ Considerações Finais

Durante o desenvolvimento, diversos desafios técnicos foram enfrentados, principalmente relacionados às animações, colisões e ajustes das fases. Embora o resultado final não represente completamente a visão inicial do projeto, a experiência foi extremamente enriquecedora e despertou ainda mais meu interesse pela área de desenvolvimento de jogos.

O projeto continuará sendo aprimorado após a entrega acadêmica, servindo também como uma peça de portfólio e como base para futuras evoluções.
